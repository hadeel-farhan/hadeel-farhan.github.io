Dubai Health Interactive Screen Game

2025


Window to Dubai Health is an interactive screen for Dubai Health employees

Designed and developed an interactive touch-screen memory game for large public displays across Dubai Health facilities to transform waiting areas into engaging educational experiences. Implemented a networked leaderboard enabling real-time competition between sites, encouraging participation and repeat play.

My Role

  • Co-designed the product concept and interaction model for a public-space digital game.
  • Built the game using JavaScript and GDevelop, optimized for large touch displays.
  • Led the product concept and UX design by creating low- and mid-fidelity wireframes, collaborating with a designer to translate them into high-fidelity UI aligned with Dubai Health’s brand.
  • Implemented game logic, scoring, and timing mechanics to support fast and engaging gameplay.
  • Created an Arabic version of the game to increase accessibility and engagement across the sites.
  • Integrated an online leaderboard system that tracks scores across Dubai Health sites.

Impact

  • Planned deployment across 31 Dubai Health sites.
  • Encourages team building and friendly cross-site competition through a shared leaderboard.
  • Turns spaces into interactive community engagement experiences.
