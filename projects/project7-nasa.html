NASA Smallsat Database Extraction

2022


Wireframes of the interface for data extraction

Developed tools to support NASA’s Commercial Smallsat Data Acquisition (CSDA) program by improving how satellite data from commercial vendors is identified, organized, and accessed. The project focused on creating a system that curates satellite data related to high-value global events, ensuring researchers can easily discover and request relevant datasets through a centralized archive.

My Role

  • Contributed to the development of a system designed to identify, curate, and organize commercial satellite data associated with high-value events.
  • Built APIs and a web interface using Django to enable user-friendly access to the STAC event database.
  • Developed an Event Search interface allowing users to query and request satellite datasets from the archive.
  • Implemented features using Python, Django, JavaScript, HTML, and CSS to connect the front-end search tool with the backend data system.

Impact

  • Improved accessibility of commercial satellite imagery data for researchers and analysts through a structured search interface.
  • Enabled users to discover and request relevant datasets more efficiently via the event-based database.
  • Contributed to the development of a robust data archive supporting NASA’s satellite data research initiatives.

Key Skills

Software Development • Python • Django • API Development • JavaScript • HTML • CSS • UI/UX Design
